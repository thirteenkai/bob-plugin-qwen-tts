# Bob Qwen TTS 插件

使用阿里云 Qwen3-TTS-Flash 模型进行语音合成的 Bob 插件。

## ✨ 功能特点

- 🎯 **高质量语音合成**: 使用 Qwen3-TTS-Flash 模型，反应速度极快
- 🌍 **多语言支持**: 中文、英文、日语、韩语等 10+ 种语言
- 🎭 **48 种语音角色**: 涵盖普通话、多地方言（北京/上海/四川/粤语等）、外语及特色音色
- 🎚️ **语速倾向**: 支持预设语速倾向和自定义语速倾向（通过 Qwen Instruct 近似控制，非官方 `speech_rate`）
- 🧠 **智能模型切换**: 自动识别音色支持的模型，确保最佳兼容性
- 🌐 **双地域支持**: 国内(北京) / 国际(新加坡)

## 📦 安装方法

### 方式一：直接下载安装（推荐）
1. 前往 [Releases](../../releases) 页面
2. 下载最新版本的 `qwen-tts.bobplugin` 文件
3. 双击该文件，Bob 会自动安装插件

### 方式二：手动安装
1. 将 `src` 文件夹内的所有文件打包为 ZIP
2. 将 ZIP 文件扩展名改为 `.bobplugin`
3. 双击安装

## ⚙️ 配置说明

### 1. 获取 API Key
1. 访问 [阿里云百炼控制台](https://dashscope.console.aliyun.com/)
2. 注册/登录后创建 API Key
3. 将 API Key 填入插件配置

### 2. 配置选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| API Key | DashScope API Key（必填） | - |
| API 地域 | 北京(国内) / 新加坡(国际) | 北京 |
| 模型 | qwen3-tts-flash / qwen3-tts-instruct-flash。旧 qwen-tts 已从菜单隐藏，仅保留历史配置兼容 | qwen3-tts-flash |
| 语音角色 | 48 种官方非实时音色可选 | Ethan |
| 语速倾向 | 预设语速倾向：0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x。仅部分 Instruct 音色支持近似控制 | 1.0 |
| 自定义语速倾向 | 输入 0.5-2.0 之间的任意值，留空使用预设语速倾向。非法值会直接报错 | - |

### 3. 语速倾向说明

当前插件使用非实时 HTTP 接口。该接口没有官方 `speech_rate` 参数。插件会在支持 `qwen3-tts-instruct-flash` 的音色上用 `instructions` 近似控制语速倾向；不支持 Instruct 的音色会保持原模型，并在用户调语速倾向时提示更换音色或恢复 1.0x。

### 4. 语音角色 (官方支持 48 种非实时音色)

插件已内置**智能模型切换**功能：旧 `qwen-tts` 已从菜单隐藏；如果历史配置仍保存为旧模型，插件会在不兼容时自动切换到 Qwen3 Flash，避免直接报错。

## 🚀 使用方法

1. 在 Bob 中进行翻译或 OCR 识别
2. 点击翻译结果旁的发音按钮 🔊
3. 即可听到 Qwen TTS 的语音朗读

## ⚠️ 注意事项

- 需要有效的 DashScope API Key
- 音频 URL 有效期为 24 小时
- 单次请求最大文本长度: 600 字符
