# Qwen TTS for Bob

Qwen TTS for Bob 是一个 Bob 文本转语音插件。它接收 Bob 需要朗读的翻译或 OCR 文本，直接调用阿里云 DashScope 的 Qwen TTS 接口，并将返回的音频交给 Bob 播放。

[下载最新版本](https://github.com/thirteenkai/bob-plugin-qwen-tts/releases/latest/download/qwen-tts.bobplugin) · [查看版本记录](https://github.com/thirteenkai/bob-plugin-qwen-tts/releases)

## 先判断它是否适合你

| 适合 | 不适合 |
|------|--------|
| 已经使用 Bob，希望改善翻译或 OCR 结果的朗读体验 | 需要完全离线、本地运行的语音合成 |
| 需要普通话、方言或多语言音色 | 不希望朗读文本发送到云端服务 |
| 可以使用自己的 DashScope API Key，并接受服务方计费 | 需要精确、逐值可复现的语速参数 |

## 它解决什么问题

Bob 负责获取和展示文本，DashScope 负责生成语音，这个插件负责连接两者，并处理实际使用中的兼容性边界：

- 根据地域、模型和音色选择可用的请求组合。
- 历史配置不再受支持时，自动回退到兼容模型。
- 对认证失败、限流、网络错误和无效响应给出明确提示。
- 仅在接口返回可用音频地址后判定合成成功。
- 在支持 Instruct 的音色上提供近似的语速倾向控制。

## 数据如何流动

1. Bob 将需要朗读的文本交给插件。
2. 插件使用你配置的 API Key，向所选地域的 DashScope 接口发起请求。
3. DashScope 返回临时音频地址。
4. 插件把音频地址交回 Bob 播放。

插件没有独立的中转服务器。朗读文本会发送到阿里云 DashScope，相关数据处理与费用遵循你的阿里云账户和服务配置。

## 快速开始

### 1. 安装插件

1. 下载最新版 [`qwen-tts.bobplugin`](https://github.com/thirteenkai/bob-plugin-qwen-tts/releases/latest/download/qwen-tts.bobplugin)。
2. 双击插件文件。
3. 在 Bob 中确认安装；已有旧版本时选择替换。

### 2. 准备 API Key

1. 打开[阿里云百炼控制台](https://dashscope.console.aliyun.com/)。
2. 登录后创建 DashScope API Key。
3. 在 Bob 的 Qwen TTS 服务配置中填写 API Key。

### 3. 完成配置

首次使用建议保持默认设置：

| 选项 | 推荐值 | 说明 |
|------|--------|------|
| API 地域 | 北京 | 必须与 API Key 可用地域一致 |
| 模型 | `qwen3-tts-flash` | 兼容范围最广的默认选择 |
| 语音角色 | Ethan | 标准普通话男声 |
| 语速倾向 | 1.0 | 不额外添加语速指令 |
| 超时时间 | 60 秒 | 弱网环境可适当增加 |

保存配置前执行一次验证。验证成功后，在 Bob 的翻译或 OCR 结果旁点击发音按钮即可朗读。

## 配置说明

| 选项 | 作用 | 默认值 |
|------|------|--------|
| API Key | DashScope 请求凭证，必填 | 无 |
| API 地域 | 选择北京或新加坡接口 | 北京 |
| 模型 | 选择 Flash 或 Instruct Flash | `qwen3-tts-flash` |
| 语音角色 | 选择朗读音色 | Ethan |
| 语速倾向 | 选择 0.5x 至 2.0x 的语速倾向 | 1.0 |
| 自定义语速倾向 | 输入 0.5 至 2.0 的值；留空使用预设值 | 空 |
| 超时时间 | HTTP 请求超时时间 | 60 秒 |

### 模型选择

- `qwen3-tts-flash` 是默认模型，适合优先考虑兼容性的用户。
- `qwen3-tts-instruct-flash` 用于支持指令控制的音色，并承担语速倾向控制。
- 旧 `qwen-tts` 已从菜单移除，仅保留历史配置兼容。地域或音色不兼容时，插件会自动切换到 `qwen3-tts-flash`。

### 语速倾向

当前插件使用非实时 HTTP 接口，该接口没有官方 `speech_rate` 参数。插件通过 `instructions` 在受支持的 Instruct 音色上近似控制语速，因此这里使用“语速倾向”，而不是“精确语速”。

如果所选音色不支持 Instruct，插件会保持兼容模型，并提示更换音色或恢复为 1.0x。

## 使用限制与排查顺序

- 单次文本最多 600 个字符，超过后插件会直接提示。
- 插件依赖网络和 DashScope 服务状态，不能离线工作。
- API Key、账户余额、接口权限和调用费用由 DashScope 管理。
- API Key 不可用时，先确认所选地域，再确认密钥状态和账户权限。
- HTTP 429 表示请求频率过高，应等待后重试。
- 音色或语言不兼容时，优先改用 `qwen3-tts-flash`，再检查音色选择。
